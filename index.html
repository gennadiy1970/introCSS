• Добавляем CSS:
    1. используя внешнюю таблицу стилей // emmet link:css
    2. <head> <style> b {color: gold;}</style></head>
    3. inline: <b style="color: gold">HELLO</b>
  • Синтаксис CSS:
    • CSS-правило: b {color: gold;}
    • селектор: b {...}
    • свойство: {color: ...}
    • значениe: { ... :gold;}
  • Селекторы:
    • любой элемент: *
    • по тегу: div
    • по классу: .wrapper
    • по псевдоклассу: :hover
    • селекторы атрибута [href="#"]
    • по id: #uniq
    • селекторы отношений:
      • потомков
        • непосредственные потомки: div>p
        • потомки на любом уровне вложенности: div p
      • селекторы нижних соседей
        • первый нижний сосед: div+p
        • все нижние соседи: div~p
